# 网吧老板模拟器

一款微信小游戏方向的轻度养成经营游戏。玩家从一家小网吧开始，通过接待顾客、升级设备、扩张经营，逐步打造更大的网咖。

## 当前阶段

项目刚完成微信小游戏原生骨架和第一版经营循环原型。

## 运行方式

1. 安装微信开发者工具。
2. 使用微信开发者工具导入本项目根目录。
3. AppID 使用：

```text
wxd37a0c27e739afba
```

4. 编译运行后，可以在模拟器中体验第一版原型。

## 开发目标

- 顾客自动进入网吧并消费。
- 玩家通过收入升级座位、电脑、网速和零食区。
- 通过扩张容量提升客流和收益。
- 后续加入随机事件、员工、任务、存档、离线收益和商业化能力。

## 协作约定

- 日常更新以 Git commit 同步到 GitHub。
- 当需要跨电脑交接时，由用户说明“今天最后一个更新”，再生成交接文档。
- 交接文档会记录完成内容、运行方式、已知问题和下一步建议。
