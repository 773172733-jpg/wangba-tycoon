# Art Direction

当前美术方向：温暖木质像素经营界面，参考 `docs/references/pixel-ui-reference.jpg`。

## 阶段策略

1. 前期不依赖贴图，用 Canvas 绘制木质面板、木地板、前台、电脑桌椅、植物和按钮图标。
2. 玩法和系统基本稳定后，再按参考图做最终贴图替换。
3. 最终贴图优先级：顶部 HUD、底部按钮、电脑桌椅、前台、员工/顾客小人、植物与墙面装饰。

## 视觉关键词

- 暖色木质边框
- 像素块边缘和深色描边
- 室内经营感，不做抽象卡片化主界面
- 前台、入口、电脑区必须一眼可识别
- UI 图标尽量用符号化像素物件，不依赖文字长标签

## 当前无贴图目标

- 顶部 HUD 接近木牌结构：店名、现金、清洁度、设置。
- 主场景接近木地板房间：厚木框、墙板、地板线、前台装饰。
- 底部操作栏接近木质按钮区：等级、分组、常用动作。
