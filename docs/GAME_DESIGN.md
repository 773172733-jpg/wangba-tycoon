# 游戏设计文档

## 核心概念

玩家经营一家网吧，通过提供上机服务和增值服务赚取现金，再将现金投入设备、空间和服务升级，吸引更多顾客。

## 第一版核心循环

```text
顾客出现 -> 占用座位 -> 产生消费 -> 顾客离开 -> 获得现金 -> 升级网吧 -> 容纳更多顾客/提升收益
```

## 初始资源

- 现金：100
- 座位：4
- 电脑等级：1
- 网速等级：1
- 零食区等级：0
- 口碑：50

## 升级方向

- 座位扩张：提高同时接待顾客数量。
- 电脑升级：提高每位顾客的基础消费。
- 网速升级：缩短顾客上机周期，提高翻台效率。
- 零食区：增加额外消费收入。

## 后续玩法方向

- 顾客类型：学生、上班族、电竞玩家、主播、包夜顾客。
- 随机事件：断网、停电、比赛日、爆款游戏上线、卫生检查。
- 员工系统：网管、保洁、收银、维修。
- 任务系统：日收入目标、接待人数、升级目标。
- 扩张系统：街边小店、社区网吧、电竞馆、连锁网咖。
- 存档系统：本地存档，后续接入微信云能力。
